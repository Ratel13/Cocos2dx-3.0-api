详细描述

UserDefault是个微型数据库，你可以将基础数据类型存储在里面或从里面读取出来.

例如:setBoolForKey("played", true)是将一个bool值存储进去, 其key是"played",因此你可以通过getBoolForKey("played")从数据库中读取该bool值

其支持的基础数据类型如下: bool, int, float, double, string

utf16字符串转换为utf8字符串

参数
参数utf16即需要进行转换的utf8字符串
参数outUtf8即转换输出的结果utf16字符串
返回
如果转换成功返回true，否则返回false
注解
在使用outUtf8之前请检查返回值 e.g.
std::string utf8;
bool ret = StringUtils::UTF16ToUTF8(u"\u4f60\u597d", utf16);
if (ret) {
do_some_thing_with_utf8(utf8);
}
